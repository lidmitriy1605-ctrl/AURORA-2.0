## 10.07.2026

### Что сделали

- Обнаружили две папки проекта: `D:\AURORA-2.0` и `D:\Git\AURORA-2.0`.
- Установили, что настоящий Git-репозиторий находится в `D:\Git\AURORA-2.0`.
- Перенесли рабочие файлы проекта в настоящий Git-репозиторий.
- Создали и отправили в GitHub коммит `fd4589a Add initial AURORA project structure`.
- Синхронизировали ветку `main` с `origin/main`.
- Установили Termux, Git и OpenSSH на смартфон iQOO 15R.
- Создали для телефона отдельный SSH-ключ GitHub.
- Настроили подключение к GitHub через `ssh.github.com` на порту `443`, поскольку SSH-порт `22` недоступен.
- Клонировали репозиторий на телефон в `~/AURORA-2.0`.
- На телефоне проверили ветку `main`, удалённый репозиторий `origin`, историю коммитов, наличие проектных документов и синхронизацию с `origin/main`.

### Какие проблемы обнаружили

- Существовали две копии проекта, из-за чего можно было продолжить работу не в настоящем Git-репозитории.
- Стандартное SSH-подключение к GitHub через порт `22` было недоступно.

### Какие решения приняли

- Основным рабочим путём назначен `D:\Git\AURORA-2.0`.
- Старая копия `D:\AURORA-2.0` не должна использоваться.
- Для телефона используется отдельный SSH-ключ GitHub и подключение через `ssh.github.com:443`.
- ПК и телефон работают с единым репозиторием `lidmitriy1605-ctrl/AURORA-2.0`.

### Текущее состояние

- Активная ветка: `main`.
- Ветка синхронизирована с `origin/main`.
- Репозиторий доступен на ПК по пути `D:\Git\AURORA-2.0` и на телефоне по пути `~/AURORA-2.0`.
- Текущий этап: Этап 1 — проектирование и подготовка проектной документации.
