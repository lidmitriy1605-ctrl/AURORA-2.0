# Журнал изменений AURORA 2.0

## 22.07.2026

- Подготовлен и включён в рабочий контур документ пользователей, ролей и областей доступа.
- Зафиксированы пользовательские сценарии, границы MVP, архитектура, модель данных, roadmap, журнал решений и протокол разработки.
- MVP ограничен локальным ядром заметок и задач без внешних интеграций.
- Начат цикл реализации технического фундамента M2.
- Реализовано локальное Python-ядро: заметки, задачи, области данных, аудит и подтверждения критических действий.
- Добавлены автоматические проверки ключевых правил MVP.
- Добавлен GitHub Actions workflow, который запускает тесты при push и pull request в `main`.
- Добавлен безопасный экспорт данных по одной разрешённой области и сценарий резервного копирования.
- Добавлен UTF-8-совместимый PowerShell-запуск для корректной работы с русским текстом.
- CLI расширен командами статусов задач, журнала и подтверждения критических действий; добавлена инструкция первого запуска.
- Запущен первый реальный пользовательский сценарий Дмитрия: заметка, задача, поиск, аудит и локальная резервная копия.
- Добавлен защищённый Telegram-адаптер с явной привязкой чата владельца и командами личного MVP.
- Telegram-интерфейс расширен естественными русскоязычными сообщениями, панелью обзора, семейными задачами, сроками и уведомлениями исполнителю.
- Подключён Open-Meteo: Telegram-бот отвечает на запросы о погоде текущими данными и прогнозом на завтра.
