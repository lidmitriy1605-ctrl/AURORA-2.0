# AURORA 2.0

Персональная интеллектуальная система для управления знаниями, задачами и контекстом Дмитрия и его семьи.

## Текущий статус

Проект проходит этап формирования MVP и технического фундамента. Официальное состояние проекта ведётся в `management/PROJECT_DASHBOARD.md`.

## Принципы

- личные данные изолированы по областям;
- критические действия всегда требуют подтверждения;
- ядро не зависит от конкретной LLM или интерфейса;
- работа ведётся маленькими, проверяемыми изменениями.

## Запуск

Технический фундамент не требует сторонних библиотек. Проверки запускаются командой:

```powershell
.\tools\verify.ps1
```

Пример локального запуска:

```powershell
$env:PYTHONPATH = 'src'
python -m aurora.cli note-add dmitry dmitry 'Идея для AURORA'
python -m aurora.cli task-list dmitry dmitry
```

Экспорт одной разрешённой области данных создаёт новую JSON-копию и никогда не перезаписывает прежнюю:

```powershell
.\tools\backup.ps1 -Actor dmitry -Space dmitry
```
