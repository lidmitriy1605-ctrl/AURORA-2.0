# 04. Архитектура системы

## Контуры

```text
Интерфейс (CLI сейчас; web/Telegram позже)
                |
          Application services
       /        |        \\
Notes       Tasks      Confirmations
       \\        |        /
            Core policies
                |
  Storage + Audit log + provider adapters
```

## Правила

- Интерфейс не обращается к хранилищу напрямую.
- Каждая операция получает контекст пользователя и область данных.
- Контроль доступа расположен в ядре, а не в интерфейсе.
- Адаптеры внешних сервисов получают только подтверждённые команды.
- Хранилище и LLM заменяемы через интерфейсы.

## Первый технический выбор

Для MVP используется Python standard library и JSON-хранилище. Это уменьшает сложность, позволяет запустить систему без установки зависимостей и не является обязательством для будущей версии.
