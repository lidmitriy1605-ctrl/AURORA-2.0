# 05. Модель данных и памяти

## Области данных

| Область | Владелец | Чтение |
| --- | --- | --- |
| `dmitry` | Дмитрий | Дмитрий |
| `eva` | Ева | Ева |
| `family` | семья | Дмитрий, Ева |

## Сущности MVP

- **Note:** `id`, `space`, `author`, `text`, `created_at`, `source`.
- **Task:** `id`, `space`, `author`, `title`, `status`, `created_at`, `updated_at`.
- **Audit event:** время, актор, действие, объект, результат.
- **Confirmation:** описание критического действия, создатель, состояние `pending` или `confirmed`.

Запись никогда не перемещается между областями автоматически. Удаление и внешние действия не входят в первое ядро.
