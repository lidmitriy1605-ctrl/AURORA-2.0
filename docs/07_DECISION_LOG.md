# 07. Журнал решений

## ADR-001 — локальный Python-фундамент

**Дата:** 22.07.2026  
**Статус:** принято для MVP

Ядро MVP реализуется на Python без сторонних зависимостей и хранит данные локально. Причина: быстрый проверяемый старт без подключения личных сервисов и без привязки к поставщику.

## ADR-002 — области данных как обязательный контекст

**Дата:** 22.07.2026  
**Статус:** принято

Каждая заметка, задача и действие привязаны к области `dmitry`, `eva` или `family`; доступ проверяется до чтения и изменения данных.

## ADR-003 — внешние действия только через подтверждение

**Дата:** 22.07.2026  
**Статус:** принято

Ядро не исполняет внешнее действие без отдельного подтверждения владельца. В MVP реализуется только безопасная подготовка такого запроса.
